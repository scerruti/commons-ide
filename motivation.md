# MOTIVATION: THE ORIGIN EMAIL

The following message was posted to a teacher community forum and is the direct catalyst for The Commons IDE project. It is preserved here as a reference artifact.

---

> Hi,
>
> As some of you may know Trinket® is going away this June. I am looking for an alternative IDE that could fit the following criteria that Trinket© offered:
>
> - **Online** - Students can save and share work online with links.
> - **Simple user interface that has minimal barriers to the beginner.** - Trinket was the best, student never had an issue getting things up and coding instantly.
> - **Low expense (if possible).** Otherwise happy to pay for the features listed here.
> - **Quick and lightweight** - Trinket has been easy to share and receive projects that boot up automatically and run quickly (even with graphics).
> - **Large Graphics output window/space for Turtle and PyGame.** Trinket provides a large window, quick generation, and instant run capabilities for turtle and pygame.
>
> Let me know if you have suggestions for alternatives.
>
> Thanks for your time.
>
> Jamie
>
> ---
> Jamie Walzl
> Pemberton BC

---

## What This Tells Us

| Jamie's Requirement | Concrete Acceptance Criterion |
|---|---|
| Online save & share | A student can generate a shareable link to their project in ≤ 2 clicks. A recipient can open and run the project with no login. |
| Zero barrier to start | A brand-new student opens the IDE and runs their first line of code in under 60 seconds, with no install, no account creation, and no tutorial required. |
| Low / no cost | No per-student fee. No subscription required from the teacher. Any recurring costs are absorbed by existing school infrastructure (Google Workspace for Education). |
| Fast boot, fast run | The IDE loads and is ready to accept code in under 5 seconds on a typical school Chromebook. Code executes and graphics appear in under 3 seconds for standard Turtle/Pygame projects. |
| Large graphics window | The graphics output panel occupies at least 50% of the visible viewport on a 1280×800 screen. Turtle and Pygame visuals render at full color without clipping. |

---

*Trinket shutdown deadline: **June 2026.** The Commons IDE must be classroom-ready before that date.*
